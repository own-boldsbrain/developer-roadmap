# AWS

A Amazon Web Services é a plataforma de computação em nuvem líder de mercado desde 2011, à frente da Azure e do Google Cloud. A AWS oferece mais de 200 serviços com data centers localizados em todo o mundo. O serviço AWS é uma plataforma online que fornece soluções de computação em nuvem escaláveis ​​e econômicas. É uma plataforma de nuvem amplamente adotada que oferece várias operações sob demanda, como poder de computação, armazenamento de banco de dados, entrega de conteúdo e assim por diante.
