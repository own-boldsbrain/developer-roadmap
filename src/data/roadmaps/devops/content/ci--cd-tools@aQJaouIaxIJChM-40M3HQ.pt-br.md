# CI/CD

CI/CD é um método para entregar aplicações aos clientes de forma frequente, introduzindo automação nas etapas do desenvolvimento da aplicação. Os principais conceitos associados ao CI/CD são integração contínua, entrega contínua e implantação contínua. O CI/CD é uma solução para os problemas que a integração de novos códigos pode causar para as equipes de desenvolvimento e operações. Especificamente, o CI/CD introduz automação contínua e monitoramento contínuo ao longo do ciclo de vida das aplicações, desde as fases de integração e teste até a entrega e implantação. Em conjunto, essas práticas interconectadas são frequentemente referidas como uma "pipeline de CI/CD" e são apoiadas por equipes de desenvolvimento e operações trabalhando em colaboração de forma ágil, utilizando abordagens DevOps ou engenharia de confiabilidade do site (SRE).
