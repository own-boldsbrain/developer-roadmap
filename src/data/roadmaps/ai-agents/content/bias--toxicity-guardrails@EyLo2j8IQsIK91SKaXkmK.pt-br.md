# Diretrizes para Prevenção de Viés e Toxicidade

As diretrizes para prevenção de viés e toxicidade impedem que um agente de IA forneça resultados injustos ou prejudiciais. O viés surge quando os dados de treinamento favorecem determinados grupos ou visões. A toxicidade é a linguagem que é odiosa, violenta ou ofensiva. Para evitar isso, comece com dados limpos e equilibrados. Remova insultos, estereótipos e spam. Adicione exemplos de várias fontes para que o modelo aprenda padrões justos. Durante o treinamento, teste o modelo frequentemente e ajuste os pesos ou regras que favoreçam uma direção. Após o treinamento, coloque filtros para bloquear palavras tóxicas ou sinalizar respostas injustas antes que os usuários as vejam. Mantenha registros, realize auditorias e solicite feedback dos usuários para identificar novos problemas precocemente. Documente cada etapa para que desenvolvedores e usuários conheçam os limites e riscos. Essas ações protegem as pessoas, cumprem as leis e ajudam os usuários a confiarem na IA.
