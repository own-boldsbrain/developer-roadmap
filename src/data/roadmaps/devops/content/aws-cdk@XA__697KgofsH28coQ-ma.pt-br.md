# AWS CDK

O AWS Cloud Development Kit (AWS CDK) é um framework de desenvolvimento de software de código aberto usado para provisionar recursos de infraestrutura na nuvem de forma segura e repetível, através do AWS CloudFormation. O AWS CDK oferece a flexibilidade de escrever infraestrutura como código em linguagens populares como JavaScript, TypeScript, Python, Java, C# e Go.
