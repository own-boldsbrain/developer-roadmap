# Desenvolvimento Básico de Backend

Antes de começar a aprender como construir agentes de IA, recomendamos que você tenha um conhecimento básico de desenvolvimento de backend. Isso inclui, conhecimento de linguagem de programação, interação com banco de dados e noções básicas de APIs, no mínimo.

Visite os seguintes recursos para obter mais informações:

- [@article@Introdução ao lado do servidor](https://developer.mozilla.org/en-US/docs/Learn/Server-side/First_steps/Introduction)
- [@article@O que é uma API REST? - Red Hat](https://www.redhat.com/en/topics/api/what-is-a-rest-api)
- [@article@O que é um Banco de Dados? - Oracle](https://www.oracle.com/database/what-is-database/)
