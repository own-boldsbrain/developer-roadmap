# ECS / Fargate

O ECS é um serviço de orquestração de contêineres que permite executar contêineres Docker em um cluster de instâncias EC2. O ECS é uma boa escolha se você deseja executar contêineres Docker em instâncias EC2 e ter controle total sobre a infraestrutura subjacente. O Fargate é um serviço de orquestração de contêineres sem servidor que permite executar contêineres Docker sem precisar gerenciar servidores, clusters ou qualquer outra infraestrutura. O Fargate é uma boa escolha se você deseja executar contêineres Docker sem precisar gerenciar servidores ou clusters.
