# Consultas ao Banco de Dados

As consultas ao banco de dados permitem que um agente de IA busque, adicione, modifique ou remova dados armazenados em um banco de dados. O agente envia uma solicitação escrita em uma linguagem de consulta, mais frequentemente SQL. O motor do banco de dados então procura pelas tabelas e retorna apenas as linhas e colunas que correspondem às regras na solicitação. Com esta ferramenta, o agente pode responder a perguntas que exigem informações atualizadas, registros de usuários ou outros fatos armazenados. Também pode criar novos registros ou ajustar os existentes para manter os dados atualizados. Como as consultas funcionam em tempo real e seguem regras claras, elas fornecem ao agente uma maneira confiável de lidar com grandes conjuntos de informações estruturadas.
