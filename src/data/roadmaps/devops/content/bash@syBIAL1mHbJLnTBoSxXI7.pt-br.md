# Scripts Bash

Bash (Bourne Again Shell) é uma poderosa shell e linguagem de comando do Unix, atuando como a shell padrão para a maioria das distribuições Linux e macOS. Ela fornece uma interface de linha de comando para interagir com o sistema operacional, executar comandos e automatizar tarefas através de scripts de shell. O Bash suporta variáveis, estruturas de controle, funções e substituição de comandos, tornando-o versátil para administração do sistema, tarefas DevOps e scripting geral. Sua capacidade de canalizar comandos, redirecionar entrada/saída e utilizar uma vasta gama de comandos e utilitários internos o torna uma ferramenta essencial para desenvolvedores e administradores de sistemas na gestão e automação de fluxos de trabalho em ambientes Unix-like.
