# Geração de código

Agentes de geração de código recebem uma solicitação em linguagem natural, compreendem o objetivo e então escrevem ou modificam o código fonte para atendê-lo. Eles podem criar pequenas aplicações, adicionar funcionalidades, corrigir bugs, refatorar código antigo, escrever testes ou traduzir código de um idioma para outro. Isso economiza tempo para os desenvolvedores, ajuda iniciantes a aprender e reduz erros humanos. As equipes usam esses agentes dentro de editores de código, ferramentas de bate-papo e pipelines automatizados. Ao lidar com tarefas de codificação rotineiras, os agentes permitem que as pessoas se concentrem no design, na lógica e nas necessidades do usuário.
