# Argo CD

O Argo CD é uma ferramenta de entrega contínua para Kubernetes, baseada na metodologia GitOps. É utilizada para automatizar o deployment e gerenciamento de aplicações nativas da nuvem, sincronizando continuamente o estado desejado da aplicação com o estado real no ambiente de produção. Em um fluxo de trabalho do Argo CD, as alterações na aplicação são feitas através do commit de código ou alterações de configuração em um repositório Git. O Argo CD monitora o repositório e automaticamente aplica as alterações ao ambiente de produção utilizando uma pipeline de entrega contínua. A pipeline é acionada por alterações no repositório Git e é responsável por construir, testar e implantar as alterações no ambiente de produção.
