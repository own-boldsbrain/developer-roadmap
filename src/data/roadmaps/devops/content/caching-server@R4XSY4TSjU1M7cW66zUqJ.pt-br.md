# Servidor de cache

Um servidor de cache, também conhecido como proxy server ou cache server, é um servidor de rede dedicado que armazena páginas da web e outros conteúdos da Internet localmente para reduzir o uso da largura de banda, a carga do servidor e a percepção de atraso. Ele funciona interceptando solicitações de clientes, salvando as respostas dos servidores web e servindo o conteúdo em cache para solicitações subsequentes da mesma informação. Os servidores de cache podem melhorar significativamente os tempos de resposta e reduzir o tráfego de rede, especialmente para conteúdos acessados com frequência. Eles são comumente usados em redes de entrega de conteúdo (CDNs), redes corporativas e provedores de serviços de Internet para otimizar o desempenho, reduzir custos e aprimorar a experiência do usuário, servindo o conteúdo de um local mais próximo do usuário final.
