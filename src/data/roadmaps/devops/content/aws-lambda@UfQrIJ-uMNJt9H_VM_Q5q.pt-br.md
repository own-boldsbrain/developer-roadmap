# AWS Lambda

A AWS Lambda é um serviço de computação sem servidor que permite executar código sem a necessidade de provisionar ou gerenciar servidores. Ela escala automaticamente, executa o código em resposta a gatilhos e cobra apenas pelo tempo de computação consumido. A Lambda suporta várias linguagens de programação e integra-se perfeitamente com outros serviços da AWS, tornando-a ideal para construir microserviços, automatizar tarefas e processar fluxos de dados com mínimo overhead operacional.
