# Git

O Git é um sistema distribuído de controle de versão projetado para rastrear alterações no código fonte durante o desenvolvimento de software. Ele permite que vários desenvolvedores trabalhem no mesmo projeto simultaneamente, mantendo um histórico completo de modificações. O Git possui repositórios locais em cada máquina do desenvolvedor, permitindo trabalho offline e operações rápidas. Ele suporta o desenvolvimento não linear por meio de ramificações e mesclagem, facilitando fluxos de trabalho paralelos. A natureza distribuída do Git melhora a colaboração, backup e experimentação. Os principais conceitos incluem commits, branches, merges e repositórios remotos. Com sua velocidade, flexibilidade e robustas capacidades de ramificação e mesclagem, o Git se tornou o padrão para controle de versão no desenvolvimento de software moderno, alimentando plataformas como GitHub e GitLab.
