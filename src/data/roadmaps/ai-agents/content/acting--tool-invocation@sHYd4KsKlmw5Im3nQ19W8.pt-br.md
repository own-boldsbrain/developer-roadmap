# Execução de Ações / Chamada de Ferramentas

A execução de ações, também chamada de invocação de ferramentas, é a etapa em que a IA escolhe uma ferramenta e a executa para obter dados do mundo real ou para realizar alguma ação. O agente analisa seu objetivo atual e o plano que acabou de criar. Em seguida, ele seleciona a melhor ferramenta, como uma pesquisa na web, uma consulta ao banco de dados ou uma calculadora. O agente preenche as entradas necessárias e envia a chamada. O sistema externo realiza o trabalho pesado e retorna um resultado. A execução de ações termina quando o agente armazena esse resultado para poder pensar no próximo passo.
