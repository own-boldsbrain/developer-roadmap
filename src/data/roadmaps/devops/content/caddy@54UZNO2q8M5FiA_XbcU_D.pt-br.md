# Caddy

O Caddy é um servidor web moderno e de código aberto, escrito em Go, projetado para ser fácil de usar, seguro e de alto desempenho. Ele possui HTTPS automático com provisionamento e renovação de certificados Let's Encrypt, suporte para HTTP/2 e HTTP/3 e uma sintaxe de configuração simples. O Caddy pode servir arquivos estáticos, atuar como proxy reverso para outros serviços e lidar com várias tarefas de hospedagem web com um mínimo de configuração. Sua arquitetura modular permite fácil extensão por meio de plugins, tornando-o versátil para diferentes cenários de implantação. A ênfase do Caddy na simplicidade e segurança o torna popular tanto para projetos pequenos quanto para ambientes de produção, especialmente entre desenvolvedores que procuram uma solução de servidor web simples e rica em recursos.
