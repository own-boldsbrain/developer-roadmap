# Execução de Código / REPL

A Execução de Código ou REPL (Read-Eval-Print Loop) permite que um agente de IA execute pequenos trechos de código sob demanda, veja o resultado imediatamente e use esse resultado para decidir o que fazer a seguir. O agente "lê" o código, "executa" em um ambiente seguro, "imprime" a saída e então retorna para mais entrada. Com essa ferramenta, o agente pode testar ideias, realizar cálculos, transformar texto, chamar APIs ou inspecionar dados sem esperar por uma construção ou implantação completa. Python, JavaScript ou até mesmo comandos de shell são escolhas comuns porque iniciam rapidamente e têm muitas bibliotecas. O feedback rápido ajuda o agente a detectar erros precocemente e refinar seu plano passo a passo. O ambiente seguro impede que o sistema hospedeiro execute ações perigosas, como excluir arquivos ou fazer chamadas de rede proibidas. Em resumo, uma ferramenta de Execução de Código / REPL fornece ao agente um local de trabalho rápido e flexível para resolução de problemas.
