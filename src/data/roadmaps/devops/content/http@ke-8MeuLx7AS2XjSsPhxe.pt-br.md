# HTTP

O HTTP (Hypertext Transfer Protocol) é a base da comunicação de dados na World Wide Web. É um protocolo de camada de aplicação que permite a transferência de vários tipos de dados, principalmente páginas web e seus componentes, entre clientes (geralmente navegadores) e servidores. O HTTP opera em um modelo de solicitação-resposta, onde os clientes enviam solicitações por recursos e os servidores respondem com os dados solicitados ou mensagens de erro. Ele é projetado para ser sem estado, o que significa que cada solicitação é independente das anteriores. O HTTP suporta vários métodos (GET, POST, PUT, DELETE, etc.) para diferentes tipos de operações. Embora originalmente tenha sido projetado para transmissão de texto simples, o HTTPS, sua versão segura com criptografia, é agora amplamente adotado para proteger dados em trânsito.
