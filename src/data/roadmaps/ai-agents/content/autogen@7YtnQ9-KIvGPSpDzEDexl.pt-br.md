# AutoGen

AutoGen é um framework Python de código aberto que o ajuda a construir agentes de IA sem começar do zero. Ele permite que você defina cada agente com um papel, objetivos e ferramentas, e então gerencia o fluxo de bate-papo entre eles e um modelo de linguagem grande, como o GPT-4. Você pode encadear vários agentes para que eles planejem, codifiquem, revisem e executem tarefas juntos. A biblioteca inclui módulos prontos para uso para memória, planejamento de tarefas, chamadas de ferramentas e execução de funções, para que você só escreva as partes exclusivas do seu aplicativo. O AutoGen se conecta ao OpenAI, Azure ou modelos locais por meio de um arquivo de configuração simples. Logs, rastreamento de custos e depuração passo a passo estão incluídos, o que facilita os testes. Como os agentes são objetos Python simples, você pode combiná-los com outras bibliotecas ou seu próprio código. O AutoGen ainda está em desenvolvimento, portanto, espere mudanças frequentes e fique atento aos custos de uso, mas é uma ótima opção quando você deseja transformar um prompt em um sistema multiagente funcional em horas em vez de semanas.
