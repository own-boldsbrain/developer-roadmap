# Kubernetes

O Kubernetes é uma plataforma de gerenciamento de contêineres de código aberto, e o produto dominante nesse espaço. Usando o Kubernetes, as equipes podem implantar imagens em vários hosts subjacentes, definindo sua disponibilidade desejada, lógica de implantação e lógica de dimensionamento em YAML. O Kubernetes evoluiu do Borg, uma plataforma interna do Google usada para provisionar e alocar recursos de computação (similar aos sistemas Autopilot e Aquaman da Microsoft Azure). A popularidade do Kubernetes o tornou uma habilidade cada vez mais importante para o Engenheiro DevOps, desencadeando a criação de equipes de Plataforma em toda a indústria. Essas equipes de engenharia de plataformas geralmente existem com o único propósito de tornar o Kubernetes acessível e utilizável para seus colegas desenvolvedores de produtos.
