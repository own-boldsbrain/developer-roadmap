# Protocolos de Rede

Os protocolos de rede são regras e procedimentos padronizados que governam como os dados são transmitidos, recebidos e interpretados em redes de computadores. Eles definem o formato, o tempo, a sequência e o controle de erros na comunicação de dados. Os principais protocolos incluem:

1. TCP/IP (Protocolo de Transmissão/Protocolo da Internet): O conjunto fundamental para a comunicação na Internet.
2. HTTP/HTTPS: Para navegação na web e transferência segura de dados na web.
3. FTP/SFTP: Protocolos de transferência de arquivos.
4. SMTP/POP3/IMAP: Protocolos de comunicação de e-mail.
5. DNS: Para resolução de nomes de domínio.
6. DHCP: Para atribuição automática de endereços IP.
7. SSL/TLS: Para comunicação segura e criptografada.
8. UDP: Para transmissão rápida e sem conexão de dados.
