# Funções do Azure

Funções do Azure é um serviço de computação sem servidor no Microsoft Azure que permite aos desenvolvedores executar código acionado por eventos sem gerenciar a infraestrutura. Suporta várias linguagens de programação e se integra perfeitamente com outros serviços do Azure, permitindo a execução escalável e sob demanda de pequenos trechos de código focados. As Funções do Azure dimensionam automaticamente com base na demanda, cobram apenas pelo tempo de computação real usado e podem ser acionadas por vários eventos, como solicitações HTTP, timers ou alterações em serviços de nuvem, tornando-o ideal para microserviços, processamento de dados e tarefas de automação de fluxos de trabalho.
