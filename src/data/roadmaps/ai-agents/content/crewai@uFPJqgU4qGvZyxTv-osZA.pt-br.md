# CrewAI

CrewAI é um framework de código aberto em Python para criar equipes de agentes de IA, chamadas de "equipe". Cada agente recebe um nome, função e um conjunto de ferramentas, e o sistema gerencia o planejamento, a comunicação e a execução entre eles. Para usá-lo, instale o pacote, defina os agentes no código, conecte-os com um objeto `Crew`, e atribua uma solicitação de missão. O CrewAI interage com um LLM como GPT-4 ou Claude, passa mensagens, executa ferramentas e retorna uma saída final. Você também pode adicionar pesquisa na web, funções personalizadas ou armazenamento de memória. Os logs são integrados para ajudar a depurar e otimizar os fluxos de trabalho.
