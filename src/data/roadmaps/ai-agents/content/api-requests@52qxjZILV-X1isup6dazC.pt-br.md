# Requisições de API

As requisições de API permitem que um agente de IA faça solicitações a outro serviço para obter dados ou executar uma ação. O agente constrói uma mensagem curta que segue as regras do serviço, envia-a pela internet e aguarda uma resposta. Por exemplo, pode chamar uma API de previsão do tempo para obter a previsão do dia ou uma API de pagamento para cobrar um cliente. Cada requisição tem um método como GET ou POST, uma URL e frequentemente um pequeno bloco de JSON com os detalhes necessários. O serviço responde com outro bloco de JSON que o agente lê e utiliza. Como as requisições de API são rápidas e claras, são uma ferramenta comum para conectar o agente a muitos outros sistemas sem esforço adicional.
